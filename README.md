# Meu Primeiro Jogo Godot

Este projeto cria um jogador "boneco palito" que se move para cima, baixo, esquerda e direita.

## Como usar

1. Abra a pasta `myfirstgame` no Godot Engine.
2. Verifique se a cena `MainScene.tscn` está configurada como cena principal.
3. Execute o jogo.

## Controles

- W / seta para cima: mover para cima
- S / seta para baixo: mover para baixo
- A / seta para esquerda: mover para a esquerda
- D / seta para direita: mover para a direita
